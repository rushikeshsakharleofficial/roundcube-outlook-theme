# Outlook Theme for Roundcube Webmail

## Overview
The Outlook theme for Roundcube Webmail provides a familiar interface for users who prefer the look and feel of Microsoft Outlook. This theme is designed to enhance the user experience by offering a clean and modern design.

## Installation
To install the Outlook theme, follow these steps:

1. Download the theme package.
2. Extract the contents to the `skins` directory of your Roundcube installation.
3. Ensure that the permissions are set correctly for the theme files.
4. Update the Roundcube configuration to use the Outlook theme by modifying the `config/config.inc.php` file:
   ```php
   $config['skin'] = 'outlook';
   ```

## Usage
Once installed, the Outlook theme will be available for users to select in their Roundcube settings. Users can switch to the Outlook theme from the preferences menu.

## Features
- Responsive design for optimal viewing on various devices.
- Customizable CSS for easy adjustments.
- JavaScript enhancements for improved interactivity.
